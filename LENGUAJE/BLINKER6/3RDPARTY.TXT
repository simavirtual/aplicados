  The vendors of the following products have reported these products to
  be compatible with Blinker. We appreciate their cooperation in
  compiling this report. This information is only as current as the date
  in the upper right hand corner and applies to the current version of
  Blinker at that date. Each product's level of compatibility with
  Blinker is listed as it was last reported by the vendor without
  verification by Blink Inc.

  If you have any questions about the level of compatibility or the
  manner in which one of these products should be linked with Blinker,
  please refer to the documentation for the product in question, or
  contact the vendor of the product.

  Many more products are compatible with Blinker than could be listed in
  this report. This is a sampling of some of the major vendors that have
  responded to our inquiries. Two alphabetic lists are attached: (I)
  compatible software products, followed by (II) their vendors.

  Note that all programs which are created by products flagged as creating
  16 bit Windows programs will ALSO run on a 32 bit Windows platform such as
  Windows 95/98 or NT. The 32 bit Windows flag below simply indicates the
  ability to create native Win32 programs.
